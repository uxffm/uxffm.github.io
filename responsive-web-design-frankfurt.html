Es spielt keine Rolle ob Sie ein iPhone 5, ein iPhone X, ein Samsung Galaxy S8 oder ein iPad (oder sogar ein Phablet nutzen). Der Inhalt sollte sich auf jedem Viewport anpassen, aer sollte fließen wie Wasser. Was ich damit meine ist, es spielt keine Rolle wie die Bildschirmauflösung und die Bildschirmbreite aussieht, die Schrift, die Bilder und der gesamte Inhalt sollte so angepasst werden, dass die Bedienung auf dem Touchscreen genau so einfach ist, wie die Bedienung mit Tastatur und Maus.

Damit das funktioniert, nutze ich bei meinem Frankfurter Responsive Webdesign Service sogenannte CSS Media Queries, auch @ Media genannt. Das ist bei der Programmierung sehr wichtig, damit auch alles auf jedem Browser glänzt.

Mobile First

Mobile First gehört zu einer oft angewendeten Technik in der Welt des Webdesigns und in der Welt der Web Entwicklung. Die meisten Webdesigner, beginnen Ihre Arbeit damit, Inhalt auf dem kleinstmöglichen Bildschirm zu erstellen und zu gestalten. Von dort aus, ist es viel leichter Bilder, Texte & Icons auf eine höhere Bildschirmauflösung anzupassen.

Ich nutze als Präprozessor SASS um das Markup für die einzelnen Seiten anzupassen. Mobile First und das Responsive Design werden immer wichtiger. Nicht Zuletzt aufgrund Google’s Mobile First Index. Der Mobile First Index bedeutet auch, dass Inhalt der nicht für Smartphone optimiert ist, in den Suchergebnissen der Suchmaschinen wie Google oder Bing, nicht mehr angezeigt wird. Das hat einen riesigen Einfluss, auf den ich auch bei meinem SEO Service für Frankfurt eingehe. Jemand der mit seiner Seite in den Suchergebnissen erscheinen möchte, hat ohne eine Smartphone-Optimierte Website schlechte Karten.
