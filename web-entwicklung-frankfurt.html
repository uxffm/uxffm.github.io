Freelance Web Entwicklung Service

Als Freelancer, biete ich für das gesamte Frankfurt am Main Gebiet schnelle, professionelle und saubere Web Entwicklung an. Ich schreibe mein Markup mit HTML5, CSS3 und programmiere mit meiner Lieblings Programmiersprache Javascript, genau genommen ES6 & PHP. PHP einer Serverseitige Programmiersprache die meistens für dynamische Webseiten welche mit Datenbanken arbeiten genutzt wird.

Es gibt nichts was ich outsource und ich arbeite nicht mit WYSIWYG Software. Als Frontend-Entwickler, kann ich eine Website mit einem Content Management System wie WordPress errichten oder sie zusammen mit Jekyll und GitHub Pages als statische Website aufbauen.

Ich stelle sicher, dass jede Seite mit allen üblichen Browsern funktioniert. Es spielt also keine Rolle ob Ihre Kunden Safari, den Internet Explorer, Chrome oder Firefox Quantum benutzen.

Ich arbeite mit dem Text Editor Sublime und halte mich an alle wichtigen Regeln des Internationalen World Wide Web Consortiums. Eine sauber programmierte Homepage ist das A und O wenn es um den Erfolg einer Website geht. Es gibt sehr viele Gestaltungsvorlagen wie das Bootstrap Framework, für alle die das Rad nicht neu erfinden möchten. Allerdings arbeite ich die meiste Zeit ohne solche Frameworks.

Ich programmiere gerne individuell angepasst an Ihre Lösungen. Für mein Cascading Style Sheets Markup nutze ich SASS als sog. Präprozessor.

Wenn Sie eine E-Commerce Lösung benötigen, programmiere ich das Front End zusammen mit WordPress und WooCommerce. WordPress eignet sich für dynamischen Webseiten hervorragend.

Wenn Sie eine statische Seite benötigen, also eine die ohne Datenbanken arbeitet, helfe ich Ihnen dabei ein GitHub Konto zu erstellen damit Sie zusammen mit Jekyll Ihre Seite kostenlos auf dem GitHub Server parken können. Damit hätte Sie keine monatlichen Hosting-Kosten und der Server von GitHub gehört zu den sichersten und zuverlässigsten der Welt.

Vorteile von statischen Websites:

  • kostenloses Premium Hosting mit GitHub Pages.
  • keine Datenbanken.
  • da man ohne Datenbanken arbeitet, ist der Sicherheit sehr hoch.
  • Sehr schnell Webseiten weil -> keine Datenbanken.
  • Die Mehrheit der Webseiten funktioniert als statische Webseite.
