Webentwicklung Frankfurt

Als Freelance SEO Manager, biete ich für das gesamte Frankfurt am Main Gebiet schnelle, professionelle und saubere Webentwicklung an. Ich schreibe mein Markup mit HTML5, CSS3 und programmiere mit meiner lieblings Programmiersprache Javascript, genau genommen ES6 & PHP. PHP ist eine Serverseitige Programmiersprache die meistens für dynamische Webseiten, welche mit Datenbanken arbeiten, genutzt wird. Eine saubere Programmierung ist das A und O einer jeden SEO Strategie.

Es gibt nichts was ich outsource und ich arbeite nicht mit WYSIWYGSoftware. Als Frontend-Entwickler, kann ich eine Website mit einem Content-Management-System wie WordPress einrichten oder sie zusammen mit Jekyll und den GitHub Pages als statische Website aufbauen. Falls Sie Hilfe mit WordPress benötigen, schauen Sie unter meinem WordPress Service nach.


vom Frameworks bis E-Commerce

Ich stelle sicher, dass jede Seite mit allen üblichen Browsern funktioniert. Es spielt also keine Rolle ob Ihre Kunden Safari, den Internet Explorer, Chrome oder Firefox Quantum benutzen.

Ich arbeite mit dem Text Editor Sublime und halte mich an alle wichtigen Regeln des Internationalen World Wide Web Consortiums. Eine sauber programmierte Homepage ist das A und O, wenn es um den Erfolg einer Website geht. Es gibt sehr viele Gestaltungsvorlagen wie das Bootstrap Framework, für alle die das Rad nicht neu erfinden möchten. Allerdings arbeite ich die meiste Zeit ohne solche Frameworks.

Ich programmiere gerne individuell angepasst an meine Lösungen. Für mein Cascading-Style-Sheet Markup nutze ich SASS als sog. Präprozessor.

Wenn Sie eine E-Commerce Lösung benötigen, programmiere ich das Front-End zusammen mit WordPress und WooCommerce. WordPress eignet sich hervorragend für dynamischen Webseiten.

Wenn Sie eine statische Seite benötigen, also eine Website die ohne Datenbanken arbeitet, helfe ich Ihnen dabei ein GitHub Konto zu erstellen, damit Sie zusammen mit Jekyll Ihre Seite kostenlos auf dem GitHub Server parken können. Damit hätten Sie keine monatlichen Hosting-Kosten und der Server von GitHub gehört zu den sichersten und zuverlässigsten der Welt.

Vorteile von statischen Webseiten:

  • kostenloses Premium Hosting mit GitHub Pages.
  • keine Datenbanken.
  • da man ohne Datenbanken arbeitet -> sehr hohe Sicherheit.
  • Sehr schnell Webseiten weil -> keine Datenbanken.
  • Die Mehrheit der Webseiten funktioniert als statische Webseite.
